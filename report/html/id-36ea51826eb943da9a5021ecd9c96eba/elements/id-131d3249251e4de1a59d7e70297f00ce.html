Pojistná událost ()
Objekt Pojistná událost reprezentuje informaci o vzniklé pojistné události. Záznam vystavuje pojišťovna a může obsahovat například informace jako je podrobnější popis toho co se stalo, rozsah poškození, výše plnění, policejní protokol v přídě že byla událost vyšetřována anebo stav likvidace pojistné události.
