Výrobní konfigurace auta ()
Objekt Výrobní konfigurace auta reprezentuje informaci o vlastnostech automobilu, tak jak jej automobilka vyrobila. Těmito informacemi mohou být identifikátory karoserie a motoru, seznam prvků výbavy automobilu, standardní i doplňkové. Výrobní konfigurací jsou myšleny i významné modifikace zásadním způsobem měnící vlastnosti automobilu. Může se jednat o nesériové úpravy pohonného ústrojí nebo rozšíření doplňkové výbavy například o parkovací senzory.
