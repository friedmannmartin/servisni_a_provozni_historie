Digitální podpis autora a časové razítko ()
Digitální podpis autora a časové razítko jsou datové objekty obsažené v každém uzlu grafu. Digitální podpis autora je unikátní pro každého uživatele, který zapisuje nový záznam. Oba využívají asymetrickou kryptografii pro zajištění důvěryhodnosti každého uzlu grafu.
