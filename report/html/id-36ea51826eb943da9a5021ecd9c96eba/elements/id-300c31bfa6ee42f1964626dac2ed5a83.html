Plný klient ()
Plný klient je komplexní aplikační komponenta, obsahující aplikační funkce Zapisování servisního záznamu a Vyhledávání záznamu. Plný klient komunikuje s databází záznamů, kam jsou zapisovány servisní záznamy. Plný klient umožňuje také v databází záznamů vyhledávat provozní a servisní záznamy.
